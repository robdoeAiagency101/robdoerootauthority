# Code of Conduct

## Our Standards

The Atmospheric Truth Layer community is built on:

- **Professionalism:** Respectful, mature communication
- **Institutional Focus:** We prioritize the work, not individual personalities
- **Transparency:** Open discussion, no hidden agendas
- **Meritocracy:** Ideas evaluated on technical merit
- **Accessibility:** Welcoming contributors of all backgrounds

## Expected Behavior

- Use professional, clear language
- Respect differing technical opinions
- Accept constructive criticism
- Focus on code and architecture, not individuals
- Attribute ideas and contributions properly

## Unacceptable Behavior

- Personal attacks or harassment
- Discrimination based on identity
- Trolling or bad-faith arguments
- Sharing others' private information
- Any behavior that disrupts collaboration

## Enforcement

Violations are addressed through:
1. Warning (first offense)
2. Temporary ban (repeated behavior)
3. Permanent removal (severe cases)

## Reporting

Report violations via GitHub Issues (private discussion option) or GitHub Discussions.

**We maintain institutional standards. Professional conduct is required.**
