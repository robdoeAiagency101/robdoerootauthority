## 📡 SYSTEM 2: NRF24 WIRELESS CAPTURE

**What it does:** Captures wireless data from NRF24 antenna and saves as legal evidence.

### Hardware (15 minutes):

1. **Get components:**
   - NRF24L01+ module (8-pin)
   - CH340 USB adapter
   - 100µF capacitor
   - Antenna
   - Jumper wires

2. **Wire it (follow diagram in guide):**
   ```
   NRF24 Pin 1 (GND)   → CH340 GND
   NRF24 Pin 2 (VCC)   → 100µF cap → CH340 3.3V
   NRF24 Pin 3 (CE)    → CH340 TX
   NRF24 Pin 4 (CSN)   → CH340 RX
   NRF24 Pin 5 (SCK)   → CH340 DTR
   NRF24 Pin 6 (MOSI)  → CH340 RTS
   NRF24 Pin 7 (MISO)  → CH340 CTS
   ```

3. **Run Python script:**
   ```bash
   python nrf24_ch340_direct.py
   ```

- ✅ Antenna data captured
