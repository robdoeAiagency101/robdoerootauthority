## 💡 How It Works

### Layer 1: Signal 🛰️
Raw satellite data from BOM, Himawari-8, GOES-16, Meteosat. Continuous atmospheric frames. Real-time sky state.

### Layer 2: Decomposition 🔐
Each frame decomposed into sub-frames (tiles). Every tile hashed with SHA256. Cryptographic fingerprint tied to source, timestamp, band, region. Change one pixel → hash completely changes.

### Layer 3: Witness ⛓️
Distributed nodes observe each tile hash. GPS-backed timestamping. HMAC-SHA256 signatures. Immutable ledger anchoring via XYO bound-witness mesh. Chain of custody: "At time T, node N witnessed tile H from satellite S."

### Result: Verified Truth ✓
Planetary grid of witnessed atmospheric data. Every tile cryptographically verified. Every moment independently confirmable. Alteration immediately detectable. Math doesn't lie.
